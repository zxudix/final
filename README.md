PRADA | Official Store Website

This is a responsive and interactive multi-page website inspired by the luxury fashion brand PRADA.
It is created using HTML, CSS, Bootstrap, JavaScript, and jQuery.
The project simulates an online store with pages for shopping, gallery, brand information, and a contact form.

Pages included:

Home (index.html) – Contains a hero section with a background image and a “Shop Now” button that redirects to the store page.
Shop (products.html) – Displays product cards with images, names, and prices. Includes a live product search, “Add to Cart” functionality, and a simple cart system.
Gallery (gallery.html) – A product gallery with category filters and a lightbox feature for viewing images.
About (about.html) – Provides information about PRADA’s history and philosophy, includes an accordion section and a product pricing table.
Contact (contact.html) – A contact form with email validation, password strength indicator, confirm password check, and a success alert message.

Main features:

Responsive design for different screen sizes.
Navigation bar with active page highlighting.
Product search and filter functionality.
Cart system with “Add”, “Edit”, and “Delete” buttons.
Form validation with email and password checks.
Password strength display (Weak, Medium, Strong).
Smooth animations and hover effects.

Technologies used:

HTML5 for structure
CSS3 for styling and responsive layout
Bootstrap 5 for design components and grid system
JavaScript (ES6) for interactive behavior
jQuery for easier DOM manipulation and animations

How to use:

Download or clone the project files.
Open index.html in any web browser.
Navigate between pages using the top menu.
Try the features on each page, such as searching products, adding to cart, or sending a message in the contact form.
